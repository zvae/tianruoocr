## 天若 OCR 文字识别开源版
> 该项目已停止维护，欢迎使用新版跨平台 OCR 工具，[树洞 OCR 文字识别](https://github.com/AnyListen/tools-ocr)：https://github.com/AnyListen/tools-ocr
